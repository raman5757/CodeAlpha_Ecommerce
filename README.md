# CodeAlpha E-Commerce

A full-stack E-Commerce web application developed using Flask and MongoDB.

## Features

- User Authentication
- Admin Dashboard
- Product Management
- Shopping Cart
- Checkout
- Orders
- User Profile
- Password Encryption
- MongoDB Atlas Database

## Technologies

- Python
- Flask
- MongoDB Atlas
- Flask-PyMongo
- Bootstrap 5
- HTML
- CSS
- JavaScript

## Installation

```bash
git clone https://github.com/yourusername/CodeAlpha_Ecommerce.git

cd CodeAlpha_Ecommerce

pip install -r requirements.txt

python app.py
```

Open

```
http://127.0.0.1:5000
```
